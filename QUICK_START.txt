STEP 1: CHECK PYTHON
====================
Open terminal/command prompt and run:
  python --version

You need Python 3.10 or higher.
If Python is not installed, download from: https://www.python.org/downloads/

STEP 2: INSTALL DEPENDENCIES
============================
In the project directory, run:
  pip install -r requirements.txt

Wait for all packages to install (takes 1-2 minutes).

================================================================================
PLATFORM-SPECIFIC NOTES
================================================================================

WINDOWS:
- Use PowerShell (not Command Prompt) for best results
- To open PowerShell: Press WIN+X, select "Windows PowerShell"
- Or use Windows Terminal: https://www.microsoft.com/en-us/p/windows-terminal/

macOS:
- If "python" doesn't work, try "python3"
- Open Terminal and run: python3 main.py

LINUX:
- Usually python3 is required: python3 main.py
- Some distributions require explicit python3 installation

================================================================================
COMMON ISSUES & QUICK FIXES
================================================================================

Issue: "python: command not found"
Fix:   Try "python3 main.py" or reinstall Python

Issue: "pip install fails"
Fix:   Run: python -m pip install --upgrade pip
       Then: pip install -r requirements.txt

Issue: "Permission denied"
Fix:   On Windows: Run PowerShell as Administrator
       On Mac/Linux: Try: python3 -m venv venv
                         source venv/bin/activate
                         pip install -r requirements.txt

Issue: "Application crashes or won't start"
Fix:   Run: pip install -r requirements.txt --force-reinstall

Issue: "ModuleNotFoundError"
Fix:   Make sure you're in the project directory
       Run: pip install -r requirements.txt

Issue: "Terminal too small" (UI display issues)
Fix:   Maximize your terminal window
       Use Windows Terminal, iTerm2, or GNOME Terminal for better support
